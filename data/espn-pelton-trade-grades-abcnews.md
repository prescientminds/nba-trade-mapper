# Summary Statistics

**Total articles found:** 28 (including the mega 2018 deadline article with 16 trades)
**Total unique trades graded:** 44
**Total team-level grades assigned:** ~100
**Date range:** August 2014 - January 2019
**Author:** Kevin Pelton (ESPN Insider)

## Grade Distribution (approximate)
- **A / A-:** ~18 instances
- **B+ / B / B-:** ~42 instances
- **C+ / C / C-:** ~22 instances
- **D+ / D / D-:** ~8 instances
- **F:** 1 instance (Bulls for Jimmy Butler trade, June 2017)

## Notable Grades
- **Only F:** Chicago Bulls for trading Jimmy Butler to Minnesota (June 2017)
- **Only D-:** Charlotte Hornets for acquiring Marco Belinelli (June 2016)
- **Highest volume single article:** 2018 Trade Deadline - 16 trades graded in one article
- **Most A grades:** Brooklyn Nets (4 times: Carroll dump, Zeller dump, Dudley trade, D'Angelo Russell acquisition)
- **Most D/F grades:** Detroit Pistons (D for Bradley trade, D for Blake Griffin trade)
